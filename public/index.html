What it does

Everything you need to master math

From basic algebra to JEE advanced — every problem solved with full context, verified answers, and animated explanations.

🤖

Multi-Agent Pipeline

Parser → Router → Solver → Verifier → Explainer agents collaborate on every problem for maximum accuracy.

👁️

LLM Vision OCR

Upload a photo of handwritten math. GPT-4o, Gemini, Claude, or Groq Vision extracts equations perfectly.

📚

RAG Knowledge Base

12-topic math corpus (algebra, calculus, vectors, probability…) with ChromaDB for context-aware solving.

🎬

Manim Visualizations

Animated step-by-step math videos generated via Manim CE — see the solution unfold visually.

✅

Solution Verification

Every answer is cross-verified with SymPy and a separate verifier agent before being shown to you.

🧠

Session Memory

SQLite + similarity search remembers your past problems and avoids redundant computation.
