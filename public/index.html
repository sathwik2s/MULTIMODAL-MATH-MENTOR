Multimodal Math Mentor

Multi-Agent Pipeline

Parser → Router → Solver → Verifier → Explainer agents collaborating on every problem.

📸

Multimodal Input

Upload a photo of handwritten math, speak your question, or type it. LLM Vision OCR handles it all.

📚

RAG Knowledge Base

12-topic math corpus with ChromaDB vector retrieval for context-aware solving and explanations.

🎦

Manim Visualizations

Animated step-by-step math videos generated via Manim CE for complex problems.

👥

Human-in-the-Loop

Uncertain answers trigger HITL review. Experts can approve, reject, or refine solutions.

📈

Session Memory

SQLite + ChromaDB similarity search remembers past problems to avoid redundant computation.
